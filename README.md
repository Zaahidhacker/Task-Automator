# Task Automator
Task automator is a __python program__ that is desgined to help you with your __day to day tasks__. It it can do really cool stuff. __Please make sure to read the important notes.__

---

## Task Automator capabilites

1. Creating a new python file for a new project
*__Needs to edit the code function__*
2. Automating E-mails
*__Make sure you have Microsoft edge pinned as the second app in the taskbar__*
3. Send a whatsapp message
*__Same as 2nd one__*
4. Search in Youtube
*__Same as 2nd one__*
5. Send a Discord messsage
*__Same as 2nd one__*
6. Automate System Health checks
*__No Customization needed__*
7. Ask questions from ChatGPT or Copilot web or Windows Copilot
*__Only works for windows 11__*

---

### Python modules to install to run Task Autoamtor

1. Pyautogui - ```pip install PyAutoGUI```  
2. Time - ```Standard library in Python```
3. Psutil - ```pip install psutil```

---

### Steps to convert to .exe
__Open terminal(cmd) and go to the directory(folder) where your python file is saved and type the following code:-__ ```python -m PyInstaller --onefile Task_Automator.py```
